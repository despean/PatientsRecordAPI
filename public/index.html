Patients Api

Below are the routes in this api

  • /patients -- get all patients
  • /patients/id -- get patient by ID
  • /patients -- add new patient
  • /patients/records/id -- add records for a given patient
  • /patients/records/id -- get records for a given patient ID
  • /patients/records/id/datatype -- get patient record for a specific data type
  • /patients/records -- get all patients records
